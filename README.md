# Java - API Rest Spring Boot

* CRUD Usuário
* CRUD Perfil
* CRUD Cidade
* CRUD Estado

### Tecnologias/ferramentas utilizadas

* Eclipse IDE
* Java 8
* Spring Boot
* Maven
* Database H2 (banco de dados em memória)
* Flyway (Migração de dados)
* Swagger (Documentação da API)
* Postman (Requisições HTTP)
